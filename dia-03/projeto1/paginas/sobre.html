Um Pouco sobre o motivo do site

Este site foi criado como parte de um projeto de desenvolvimento web, com o objetivo de praticar HTML, CSS e JavaScript. O tema escolhido foi Jujutsu Kaisen por ser um anime com uma estética marcante, personagens fortes e uma narrativa envolvente. A ideia é usar esse universo como base para criar páginas com diferentes seções, aplicar estilos personalizados e integrar elementos visuais e interativos. Todas as informações, imagens e vídeos utilizados aqui são apenas para fins educacionais e não têm fins comerciais. O projeto ainda está em construção e serve como portfólio e exercício de aprendizado. Patrocinado pela instituição Grau Tecnico.
